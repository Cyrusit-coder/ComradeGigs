Quick Study Guide

1. Standard Dimensions
  • Instagram Post: 1080 x 1080 px
  • Instagram Story: 1080 x 1920 px
  • A4 Poster: 210 x 297 mm
2. Color Contrast

Always ensure text is readable. Dark text on light backgrounds, or light text on dark backgrounds. Avoid yellow text on white!

3. Tools to Use

You don't need Photoshop. Use Canva (free version) for most comrade gigs. It has templates for everything.

4. Deliverables

Clients usually want a JPG/PNG for social media and a PDF for printing. Always ask before starting.

The Practical Test
